# News Module

## Description

This module allows you to publish news stories to any Space -  you can upload a photo and select a layout.

### Version 
1.0

### Features:

- Text Editor Facility
- News With Images
- Layouts Selection
- Sidebar Widget For Latest News
- Change Authors Of The News Stories
